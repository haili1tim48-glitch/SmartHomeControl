# Technical Skills: Gesture Control App Project

## 1. UI Navigation & State Management
- **Skill:** Implementing `Spinner` widgets with custom string arrays from `strings.xml`.
- **Skill:** Managing `Intent` extras to pass "Gesture Labels" (e.g., `LightOn`) between Screen 1, Screen 2, and Screen 3.

## 2. Media Handling
- **Skill:** Using `VideoView` or `ExoPlayer` to display expert videos from the `Expert-Gestures.zip`.
- **Skill:** Implementing logic to allow replaying videos at least 3 times.

## 3. Camera Integration
- **Skill:** Utilizing the `Camera2` API or `CameraX` to record 5-second video clips using the front camera.
- **Skill:** Programmatic control of recording duration (exactly 5 seconds).

## 4. File & Network Operations
- **Skill:** Formatting files according to the schema: `[GESTURE NAME]_PRACTICE_[practice number]_[USER LASTNAME].mp4`.
- **Skill:** Implementing `Multipart` HTTP requests to upload `.mp4` files to a local Flask server.

## 5. Automated Testing
- **Skill:** Writing Espresso tests to simulate user selection, button clicks, and camera intent verification.
